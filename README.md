# TinyNotes
A lightweight app for writing and managing notes, built with:
- React
- Serverless
- JavaScript/JSX
- CSS3
- AWS:
   * S3
   * Cloud Front
   * Route 53
   * Certificate Manager
   * Cloud Watch
   * DynamoDB
   * Cognito
   * IAM

<https://ssbnotes.com>

## Installation
1. Fork and clone
2. In the notes-app-client folder, run ```npm start```
3. Open a browser to ```http://localhost:3000``` to see the app
